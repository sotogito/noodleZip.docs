| 위치                          | 무슨 예외?                                               | 어떻게 처리?                                       |
| --------------------------- | ---------------------------------------------------- | --------------------------------------------- |
| **컨트롤러 (`BindingResult`)**  | ❌ 스프링이 `@Valid/@Validated` 로 DTO 필드 단위 유효성 검사에 걸린 오류 | → `bindingResult.hasErrors()` 로 바로 if 체크 후 처리 |
| **서비스 (`CustomException`)** | ❌ 비즈니스 로직(중복 회원 가입 등)에서 직접 던진 예외                     | → `@ControllerAdvice`에서 잡아서 JSON/화면 응답        |
